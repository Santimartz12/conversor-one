# Challenge ONE - _Conversor de Divisas en Java_

En esta oportunidad, a los Devs se nos solicitó crear un conversor de divisas utilizando el lenguaje Java. 
Las características solicitadas por nuestro cliente son las siguientes:

## El conversor de moneda debe:

- Convertir del peso Colombiano a Dólar
- Convertir del peso Colombiano a Euros
- Convertir del peso Colombiano a Libras Esterlinas
- Convertir del peso Colombiano a Yen Japonés
- Convertir del peso Colombiano a Won sul-coreano
- Convertir de forma inversa con cada divisa

## Extra

Como extra al desafio decidimos agregar un conversor de temperatura. El cual permite la conversión entre los distintos tipos de grados (Fahrenheit, Celsius y Kelvin)
